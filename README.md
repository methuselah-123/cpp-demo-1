# C++ K-Means Clustering from Scratch

A modern C++ implementation of k-means clustering using Eigen.

## Features
- Synthetic dataset generation
- Unsupervised learning (k-means)
- Performance timing
- Clean C++ architecture

## Build Instructions

### Requirements
- C++17 compiler
- CMake
- Eigen3

### Build & Run
```bash
git clone https://github.com/yourname/cpp-kmeans.git
cd cpp-kmeans
mkdir build
cd build
cmake ..
make
./kmeans
